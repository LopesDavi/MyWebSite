Tecnologias que tenho experiência:

Node.js

O Node.js pode ser definido como um ambiente de execução Javascript server-side. Através do Node.js é possível criar aplicações Javascript em uma máquina sem depender de um navegador (browser) para execução.
(Utilizado pela Netflix, Uber, Linkedln, etc...)

Javascript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web, com JavaScript é possível criar conteúdos que são atualizados dinamicamente, controlar múltimídias, imagens animadas, e tudo o mais que há de intessante.
(Utilizado pelo Google, Microsoft, PayPal, etc...)

React JS

React JS é uma biblioteca JavaScript para a criação de interfaces de usuário — ou UI (user interface). React divide uma tela em diversos componentes para, então, trabalhar sobre eles de maneira individual. Isso torna o React uma tecnologia muito flexível para a solução de problemas e para a construção de interfaces reutilizáveis, uma vez que cada um destes componentes pode ser manipulado de maneira distinta.
(Utilizado pelo Walmart, Instagram, Airbnb, etc...)

Typescript

TypeScript é um superconjunto de JavaScript, ou seja, um conjunto de ferramentas e formas mais eficientes de escrever código JavaScript, adicionando recursos que não estão presentes de maneira nativa na linguagem.
(Utilizado pelo Canva, N26, DoorDash, etc...)

HTML5

HTML é uma das linguagens que utilizamos para desenvolver websites. O acrônimo HTML vem do inglês e significa Hypertext Markup Language ou em português Linguagem de Marcação de Hipertexto. Trata-se de uma linguagem de marcação para desenvolver páginas e documentos eletrônicos para a internet, fornecendo informações para usuários, navegadores e mecanismos de busca.

CSS3

CSS (Cascading Style Sheet) é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site, ou seja, permite que as mesmas marcações de uma página sejam apresentadas em diferentes estilos.

NPM

O NPM (Node Package Manager) é um poderoso gerenciador de pacotes utilizado para administrar as bibliotecas e frameworks utilizados em uma aplicação. Trata-se de uma ferramenta crucial para realizar um desenvolvimento organizado e otimizado, pois permite instalar dependências em um projeto, mantê-las atualizadas e muito mais.

SASS

O SASS (Syntactically Awesome Style Sheets) é uma linguagem de extensão do CSS. Sua ideia é adicionar recursos especiais como variáveis, mixins, funções, operações e tornar o processo de desenvolvimento mais simples e eficiente.

GIT

O GIT é um sistema de controle de versões distribuído. Estes sistemas de controle possuem a função de registrar quaisquer alterações feitas em cima de um código, armazenando essas informações e permitindo que, caso seja necessário, um(a) programador(a) possa regredir a versões anteriores de uma aplicação de modo simples e rápido. Esse tipo de sistema também simplifica muito o processo de compartilhamento de um projeto com um time, por exemplo, ou com outros(as) programadores(as).

Jquery

Trata-se de uma biblioteca JavaScript que pode ser adicionada aos projetos de codificação. Basicamente, o jQuery permite que os desenvolvedores web conectem recursos de rotina de JavaScript em uma página da web.

Bootstrap

O Bootstrap é um framework front-end gratuito e de código-fonte aberto, que disponibiliza componentes prontos para utilizar em projetos. Com ele, é possível criar e personalizar sites responsivos para dispositivos móveis, desktops e notebooks, com componentes pré-construídos e com plugins JavaScript poderosos.

Webpack

Webpack é um "empacotador de códigos" voltados para projetos da web, focando em módulos de sua aplicação. Os módulos apresentados pelo programa são estáticos e se voltam para JavaScript moderno. Assim que todas as aplicações são processadas, será possível gerar um gráfico capaz de mapear cada um dos módulos, bem como suas dependências. Isso faz com que um ou mais pacotes sejam gerados.

Babel

O Babel é um transpilador JavaScript. Entre as suas vantagens estão ter código aberto e ser multiplataforma, permitindo assim o acesso fácil de todo desenvolvedor que precisa fazer essa alteração.

JSON

JSON (JavaScript Object Notation) é basicamente um formato leve de troca de informações/dados entre sistemas. O JSON além de ser um formato leve para troca de dados é também muito simples de ler.

Mysql

O MySQL é um sistema gerenciador de banco de dados relacional de código aberto usado na maioria das aplicações gratuitas para gerir suas bases de dados. O MySQL utiliza a linguagem SQL (Structure Query Language), que é a linguagem mais popular para inserir, acessar e gerenciar o conteúdo armazenado num banco de dados.

Python

Python é uma linguagem de programação de alto nível, dinâmica, interpretada, modular, multiplataforma e orientada a objetos — uma forma específica de organizar softwares onde, a grosso modo, os procedimentos estão submetidos às classes, o que possibilita maior controle e estabilidade de códigos para projetos de grandes proporções.
